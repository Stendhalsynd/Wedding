# S3-14 Compare Numeric Sort Accuracy

## 목표
- 비교 화면에서 대관료와 식대가 현재 선택된 홀 값 기준으로 정확히 정렬되어야 한다.
- 상태 행은 정렬 대상에서 제외한다.

## 구현 규칙
- `Compare.tsx`는 대관료/식대 정렬 시 top-level hall 문서 값이 아니라 현재 선택된 hall slot 값을 사용한다.
- `상태` 행은 `sortable: false`로 두고 sort icon, active style, 클릭 동작 대상에서 제외한다.

## 검증 기준
- `Compare.tsx`가 `rentalFee`, `mealFee`에 대해 selected hall 값을 비교 기준으로 사용한다.
- `Compare.tsx`에서 상태 row는 `sortable: false`다.
- `npm test -- tests/unit/compare-sorting-behavior.test.ts`가 통과한다.
