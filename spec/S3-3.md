# S3-3 Production Domain Alignment

## 목표
- README에 기재된 운영 도메인을 실제 운영 도메인과 일치시킨다.
- 최신 변경사항을 `wedding-jwjh.vercel.app` production 도메인에 다시 배포한다.

## 구현 규칙
- 문서의 운영 URL은 더 이상 이전 Vercel 임시 도메인을 가리키면 안 된다.
- 배포는 Vercel CLI의 production 배포로 수행한다.
- 배포 후 최종 production URL을 확인한다.

## 검증 기준
- `README.md`에 `wedding-jwjh.vercel.app`가 기재된다.
- Vercel 배포가 성공한다.
- 배포 결과에서 production URL이 `wedding-jwjh.vercel.app`로 확인된다.

## 제외 범위
- GitHub Release/APK 업로드
- 커스텀 도메인 DNS 설정 변경
