# S3-10 Fixed Tab Headers And Readable Android Status Bar

## 목표
- 투어, 지도, 비교 화면의 제목 영역이 스크롤과 무관하게 화면 상단에 고정되어 보이게 한다.
- Android status bar 영역은 아이콘/텍스트가 배경과 충분히 대비되도록 한다.

## 원인
- 기존 sticky header는 실제 Android 앱 스크롤 구조에서 기대대로 고정되지 않는다.
- status bar 배경만 어둡게 바뀌고 foreground style이 사용자 환경에서 충분히 반영되지 않아 아이콘이 묻힌다.

## 구현 규칙
- 투어, 지도, 비교 화면 타이틀은 fixed header 구조를 사용한다.
- 본문은 고정 헤더 높이만큼 상단 패딩을 가져야 한다.
- Android status bar는 밝은 배경 + 어두운 아이콘 조합으로 맞춘다.
- HallDetail 상단 뒤로가기 헤더는 safe-top spacing을 유지한다.

## 검증 기준
- `Dashboard.tsx`, `Map.tsx`, `Compare.tsx`가 `fixed inset-x-0 top-0` 헤더 구조를 가진다.
- `systemUi.ts`, `capacitor.config.ts`, Android theme 색상이 밝은 status bar 기준으로 설정된다.
- `npm test -- tests/unit/android-system-ui.test.ts tests/unit/sticky-screen-headers.test.ts`가 통과한다.
