# S3-11 Map Stretch And Bottom Detail Dock

## 목표
- 지도 화면에서 실제 지도가 남는 공간을 최대한 채우도록 한다.
- 웨딩홀 이름/주소/홀 이름/예식시간 영역은 하단 탭바 바로 위에 붙어서 보이게 한다.

## 원인
- 현재 지도 컨테이너는 늘어난 높이를 iframe이 온전히 채우지 못해 하단에 빈 영역이 생긴다.
- 상세 카드가 지도 아래 일반 문서 흐름에 있어 화면 하단을 넘어가거나 탭바와 분리되어 보인다.

## 구현 규칙
- 지도 iframe은 절대 배치 또는 동등한 방식으로 컨테이너 전체를 채워야 한다.
- 지도 본문 영역은 남는 공간을 `stretch`처럼 채우는 flex 구조여야 한다.
- 상세 카드는 지도 영역 하단에 dock된 형태로, 하단 탭바 상단에 붙어 보여야 한다.

## 검증 기준
- `Map.tsx`에서 iframe이 `absolute inset-0` 기반으로 전체 높이를 채운다.
- `Map.tsx`의 상세 카드가 지도 영역 하단 dock 구조를 가진다.
- `npm test -- tests/unit/map-view-scale.test.ts`가 통과한다.
